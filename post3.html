How to Build a Blog Website Using HTML, CSS, Host it on GitHub, and Deploy it to Vercel

Step 1: Set Up Your Development Environment

Before we start, ensure you have the following tools:

  • Text Editor: Use a code editor like Visual Studio Code.
  • Git: Install Git on your computer.
  • GitHub Account: Create an account at GitHub if you don’t already have one.

Step 3: Host Your Blog on GitHub

1. Create a GitHub Repository

Log in to your GitHub account and follow these steps:

  • Click the New Repository button.
  • Name your repository (e.g., my-blog) and keep it public.
  • Click Create Repository.

2. Upload Your Files

You can upload your files in two ways:

  • Option 1: Using the Web Interface
    Click Add File > Upload Files. Drag and drop your index.html and style.css files. Click Commit Changes to save them.
  • Option 2: Using Git (Command Line)
    git init
    git add .
    git commit -m "initial commit"
    git remote add origin https://github.com/yourusername/my-blog.git
    git push -u origin main

Step 4: Deploy Your Blog on Vercel

Follow these steps to deploy your blog:

  • Create a Vercel account if you haven’t.
  • Sign up with your GitHub account.
  • Click New Project and connect your GitHub repository.
  • Click Deploy and wait for the process to complete.
  • Once deployed, Vercel will provide you with a live URL for your website.

With these steps, you can successfully build, host, and deploy your blog website!
